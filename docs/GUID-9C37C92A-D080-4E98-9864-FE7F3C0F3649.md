# CLASSB_SRAM_MARCH_ALGO

**Summary**

Selects the RAM March algorithm to run.

**Description**

Selects the RAM March algorithm to be used for the SRAM self-test.

**Remarks**

None.

```c
typedef enum
{
CLASSB_SRAM_MARCH_C = 0,
CLASSB_SRAM_MARCH_C_MINUS = 1,
CLASSB_SRAM_MARCH_B = 2
} CLASSB_SRAM_MARCH_ALGO;
```
